# Iris_Flower_Classification

This project is part of the Oasis Infobyte / ACITE Data Science Internship.

## Project Description
The goal of this project is to classify iris flowers into three species: Setosa, Versicolor, and Virginica using machine learning.

## Dataset
The built-in Iris dataset from the sklearn library is used.
No external dataset download is required.

## Algorithm used
Logistic Regression

## Tools & Technologies
- Python
- Google Colab
- Pandas
- Scikit-learn

## Result
The model acheived good accuracy in predicting the flower species.
